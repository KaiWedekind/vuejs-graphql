# Vue.js-Apollo-GraphQL

### Install Dependencies

**Application**
```
cd app
```
```
npm install
```

**Server**
```
cd server
```
```
npm install
```

### Start Server
```
cd server
```
```
npm start
```

Listening at http://localhost:3000


### Start Application
```
cd app
```
```
npm run dev
```

Listening at http://localhost:9000
